**Data types**
each column in mysql database table requires a name and a data type 
user have to specify what type of data will be stored in that perticular column when creating a table . 
in mysql there are tree main data types 
- string 
- numeric 
- date and time 

string data type 
	 - CHAR(size) = a fixed length string (contians letters, numbers and special characters ) the size parameter specifies the length of the column in chars form 0 to 255 . default is 1 
	 - VARCHAR(size) = a variable length string (contian letter number and special char ) the size '''' can be from 0 to 65535
	 - BINARY(size) = equal to char()
